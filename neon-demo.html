How to Use

Adding Neon Text Effects

Simply add the neon class to any heading or text element:

<h1 class="neon-glow">Glowing Title</h1>
<h2 class="neon-purple">Purple Heading</h2>
<h3 class="neon-pink">Pink Heading</h3>
<p class="neon-subtle">Subtle glow text</p>
<span class="neon-rainbow">Rainbow text</span>
<div class="neon-outline">Outline effect</div>
